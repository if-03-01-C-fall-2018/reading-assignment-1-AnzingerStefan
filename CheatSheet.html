A: They are addressed with their own URL for example http://www.example.com/index.html

Web Server

1. What is a web server?

A: Web server refers to server software, or hardware dedicated to running said software, that can serve contents to the World Wide Web


2. What does a web server?

A: A web server stores, processes and delivers web pages to clients.


3. How can content be stored on a web server?

A: typically, it can be stored as a real file on the servers secondary storage.


4. How is content in webpages described?

A: Content is described as style sheets, scripts, and images, to present the web page
